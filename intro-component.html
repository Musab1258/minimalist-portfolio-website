Project Background

This project was a front-end challenge from Frontend mentor. The site is designed to be fully responsive and aesthetically organized for mobile and desktop view. I used Flexbox for the mobile and Grid to create the desktop layout. I used javascript form validation API to validate the user's input from the form and send an error response when the input is empty or its format and length is not correct.
